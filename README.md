🚀 Node Social App

A full-stack social media application built with Node.js, Express, MongoDB, and modern JavaScript practices.
This project demonstrates real-world backend architecture including authentication, user management, and post creation.

✨ Features
🔐 User Authentication (JWT-based)
👤 User Profiles
📝 Create & Manage Posts
🍪 Secure Cookies Handling
🔒 Password Hashing (bcrypt)
⚡ RESTful API Structure
🛠️ Tech Stack
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
Authentication: JWT (JSON Web Tokens)
Security: bcrypt, cookie-parser
Templating: EJS
📂 Project Structure
/models        → Database schemas  
/routes        → Application routes  
/views         → EJS templates  
/public        → Static files  
app.js         → Main server file  
⚙️ Installation & Setup
